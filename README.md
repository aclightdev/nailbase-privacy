# nailbase-privacy

Публичная документация приложения [NailBase](https://github.com/aclightdev/nailbase-app):
политика конфиденциальности, инструкция пользователя, гид для тестировщиков.

Опубликовано через GitHub Pages: **https://aclightdev.github.io/nailbase-privacy/**

Политика конфиденциальности (`privacy-policy.md`) — не редактируется здесь
руками, а копируется из единственного источника истины
[`app/src/main/assets/privacy_policy.md`](https://github.com/aclightdev/nailbase-app/blob/main/app/src/main/assets/privacy_policy.md)
в репозитории `nailbase-app` скриптом
[`scripts/sync_privacy_policy.sh`](https://github.com/aclightdev/nailbase-app/blob/main/scripts/sync_privacy_policy.sh).
