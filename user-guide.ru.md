# Инструкция пользователя NailBase

*[English](user-guide.en.md) · [Українська](user-guide.uk.md)*

NailBase — офлайн-приложение для мастеров и салонов маникюра: клиенты, визиты,
отчётность. Все данные хранятся только на вашем устройстве, если вы сами не
включите синхронизацию с Google Drive (см. раздел «Резервное копирование»).

## 1. Профиль мастера и бизнес-карта

Настройки → **Профиль мастера**: имя, телефон, адрес, часы работы (можно
включить режим «Круглосуточно» вместо конкретных часов). Эти данные
используются в бизнес-карте и в тексте реферального приглашения клиенту.

Настройки → **Бизнес-карта** — визитка с вашими контактами, которой можно
поделиться (кнопка «Поделиться» открывает системное меню — мессенджер,
почта и т.п.). Логотип, акцентный цвет, шрифт и слоган карты настраиваются
там же, но кастомизация — Pro-функция (см. раздел 6); имя/телефон/адрес/часы
на карте всегда берутся из профиля мастера и правятся бесплатно.

## 2. Клиенты и визиты — базовый флоу

1. На экране клиентов добавьте нового клиента (имя, телефон — можно
   импортировать из контактов телефона одним касанием) или откройте
   существующего.
2. На карточке клиента нажмите «Добавить визит», выберите услуги — их
   длительность и суммарная стоимость посчитаются автоматически.
3. Визит появится на циферблате дня в виде цветного сектора; статус визита
   (ожидает, состоялся, не пришёл, аннулирован) отмечается вручную после
   визита — приложение не проставляет «Состоялся» само по факту истекшего
   времени.
4. Если новое время визита пересекается с уже существующим, приложение
   предупредит об этом до сохранения, показав, чей это визит и на сколько
   минут пересечение.

## 3. Реферальная система — пригласить клиента

На карточке клиента — кнопка **«Показать QR-код для приглашения»**. Это
персональный QR-код именно этого клиента: отсканировавший его новый клиент
автоматически будет отмечен как приведённый им. Кнопка **«Поделиться»**
отправляет и QR-картинку, и готовый текст приглашения (с вашими контактами
из профиля мастера) через системное меню — например, в мессенджер клиента.

Тиры реферальной программы (сколько друзей нужно привести для скидки) и
накопительный бонус за постоянство настраиваются в Настройки → **Реферальная
программа** — это бесплатная функция, доступна без Pro.

## 4. Резервное копирование

Настройки → **Данные**:

- **Создать резервную копию** — собирает базу данных и фото клиентов/бизнес-
  карты в один ZIP-файл и открывает системное меню «Поделиться»: сохраните
  его туда, куда хотите (облако, компьютер, мессенджер самому себе). Файл
  нигде не загружается автоматически — только через ваш собственный выбор.
- **Восстановить из резервной копии** — выберите ранее созданный ZIP-файл.
  **Внимание:** это полностью заменяет текущие данные в приложении,
  отменить нельзя — приложение попросит подтверждение перед восстановлением.

Отдельно — **Синхронизация с Google Drive**: автоматический фоновый бэкап в
приватную папку вашего собственного Google-аккаунта (не видна в обычном
списке файлов Drive, недоступна ни нам, ни кому-либо ещё). По умолчанию
выключена. Это Pro-функция.

## 5. Отчёт для владельца салона

Если вы работаете в салоне и должны показывать выручку/отчисления
владельцу, экран **«Отчёт для салона»** (доступен из раздела «Отчётность»)
считает доход за выбранный период и сумму к оплате по проценту комиссии
салона, и превращает это в QR-код («Сгенерировать QR»).

Передача — «из рук в руки», без интернета: покажите QR-код на своём экране,
владелец салона сканирует его **своим** телефоном в NailBase (экран «Отчёты
мастеров», роль «Владелец салона», кнопка сканирования) — так отчёт
появляется в его списке отчётов по всем мастерам, без пересылки файлов и
без сервера.

## 6. Free и Pro

Free-версия ограничена **150 клиентами и 600 визитами** — этого достаточно
для полноценной работы небольшого мастера; при достижении лимита
приложение предложит перейти на Pro. Дополнительно к Pro относятся:

- расширенная статистика (график выручки, разбивка по категориям услуг, топ
  рефереров);
- кастомизация бизнес-карты (логотип, акцентный цвет, шрифт);
- синхронизация с Google Drive и напоминания клиентам — в разработке,
  появятся позже.

**Важно:** на текущей версии покупка Pro в самом приложении технически ещё
не подключена (биллинг настраивается) — кнопка «Купить» на экране Pro пока
показывает «Скоро будет доступно». Как только покупка заработает, эта
инструкция будет обновлена.

## 7. Смена языка и валюты

Настройки → **Основное**: **Язык** — можно оставить «Системный язык»
устройства или выбрать один из поддерживаемых языков интерфейса вручную;
**Валюта** — выбор из полного списка ISO 4217 по коду или названию (поиском)
либо ручной ввод произвольного символа через «Другая валюта», если нужной
нет в списке. Оба переключаются мгновенно, без перезапуска приложения.
